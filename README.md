# Выравнивание колонок по высоте

## Установка

```html
<!-- Подключаем библиотеку jQuery (если еще не подключена) -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<!-- Подключаем плагин jquery.colHeight.js -->
<script src="/js/jquery.colHeight.js"></script>
<!-- Подключаем скрипт со следующим кодом либо пишем непосредственно в нужном файле: -->
<script src="/js/yourScript.js"></script>
```
```html
	<!-- для выравнивания при первоначальной загрузке -->
	$( window ).load(function() {
	  $(".класс общей обертки колонок").colHeight({class:"класс одной колонки", minWidth:  "768"});
	});

	<!-- для выравнивания при изменении ширины экрана -->
	$( window ).resize(function() {
	  $(".класс общей обертки колонок").colHeight({class:"класс одной колонки", minWidth:  "768", event:"resize"});
	});
```

## Параметры плагина:

```
class:   класс одной колонки
minWidth: минимальная ширина, при которой будет срабатывать выравнивание колонок
event:   событие, при котором будет вызвано выравнивание колонок. По умолчанию ready.
	- ready: при первоначальной загрузке
	- resize: при изменении ширины экрана
```
